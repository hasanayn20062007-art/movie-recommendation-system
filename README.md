# Movie Recommendation System

This project is a content-based movie recommendation system built using Python and NLP techniques.

## Technologies Used
- Pandas
- NumPy
- Scikit-learn
- NLP
- CountVectorizer
- Cosine Similarity

## Features
- Recommends similar movies
- Uses movie tags and overview
- NLP preprocessing
